핵심 기술

LangGraph StateGraph 설계

State 정의

class PipelineState(TypedDict):
    # 입력
    job_id: str
    video_path: str
    use_ai_thumbnail: bool
    jobs: Any       # Job dict 뮤터블 참조

    # 파이프라인 데이터
    duration: float
    audio_path: str
    segments: list   # Whisper 세그먼트
    moments: list    # Gemini 선정 장면

    # Fan-out 누적 채널
    results: Annotated[list, operator.add]

Annotated[list, operator.add]
fan-out된 node_process_clip N개가 각각 {"results": [short]}를 반환하면 LangGraph가 자동으로 리스트를 누적합니다.

그래프 빌드

def _build_graph():
    g = StateGraph(PipelineState)

    # 노드 등록
    g.add_node("node_transcribe", node_transcribe)
    g.add_node("node_select_moments", node_select_moments)
    g.add_node("node_process_clip", node_process_clip)
    # ... 나머지 노드

    # 순차 엣지
    g.add_edge(START, "node_get_duration")
    g.add_edge("node_get_duration", "node_extract_audio")
    # ...

    # Fan-out 분기
    g.add_conditional_edges(
        "node_select_moments",
        node_dispatch_clips,   # Send() 반환
        ["node_process_clip"],
    )
    g.add_edge("node_process_clip", "node_concat_highlight")

    return g.compile()

Send() Fan-out 패턴

def node_dispatch_clips(state: PipelineState):
    """핵심 장면 N개 → node_process_clip N개를 동시에 분기"""
    return [
        Send("node_process_clip", {
            "moment": m,
            "video_path": state["video_path"],
            "segments": state["segments"],
            "use_ai_thumbnail": state["use_ai_thumbnail"],
        })
        for m in state["moments"]   # 장면 수만큼 병렬 실행
    ]

각 노드는 async def로 구현되어 await _graph.ainvoke(...)로 전체 파이프라인이 비동기 실행됩니다. FastAPI BackgroundTasks에 등록되어 업로드 응답은 즉시 반환되고 진행 상황은 SSE로 스트리밍됩니다.

① 인터뷰 단계

4개 고정 질문(타겟/톤/핵심메시지/길이)으로 GPT-4o에 전달할 컨텍스트를 수집합니다. Stateless API — 클라이언트가 answers 배열을 누적해 전송합니다.

② 스크립트 생성

GPT-4o가 response_format: json_object로 장면별 narration(한국어) + image_prompt(영어)를 반환합니다. 첫 장면은 훅, 마지막은 CTA 구조가 프롬프트에 강제됩니다.

③ 병렬 생성

각 장면의 TTS(tts-1-hd)와 gpt-image-1 이미지를 asyncio.gather로 동시에 생성합니다. TTS 오디오 길이를 ffprobe로 측정해 이미지 루프 영상의 길이를 맞춥니다.

📡 SSE 실시간 진행률 스트리밍

각 LangGraph 노드가 시작될 때 jobs dict 참조로 step / progress를 업데이트합니다. 클라이언트는 SSE 엔드포인트를 구독해 1초 간격으로 수신합니다.

# FastAPI SSE 엔드포인트
async def event_stream():
    while True:
        data = job.model_dump_json()
        yield f"data: {data}\n\n"
        if job.status in (DONE, ERROR):
            break
        await asyncio.sleep(1)

🎬 하이라이트 프레임 캡처

처리된 9:16 클립이 아닌 원본 영상에서 직접 프레임을 추출합니다. 장면 중간 지점(start + duration × 0.5)을 ffmpeg -ss seek으로 캡처해 preview_N.jpg로 저장합니다.

프런트엔드에서 AI 썸네일 ↔ 원본 프레임을 토글해 비교할 수 있습니다.

✍️ ASS 자막 번인 (추가 AI 없음)

Whisper 세그먼트의 타임스탬프를 클립 기준 상대 시간으로 변환해 ASS 파일을 직접 생성합니다. TikTok/Reels 스타일 (Malgun Gothic 82pt, 흰색+검정 테두리)로 ffmpeg libass가 인코딩 중 번인합니다.

# 클립 구간 외 세그먼트 필터링 + 시간 보정
relevant = [{
  "start": max(0, seg.start - clip_start),
  "end":   min(clip_end - clip_start, seg.end - clip_start),
  "text":  seg.text,
} for seg in segments if seg.end > clip_start]

⚡ asyncio 비동기 설계

모든 I/O 바운드 작업 — ffmpeg subprocess, OpenAI API 호출, Gemini API — 은 asyncio.to_thread()로 이벤트 루프를 블로킹하지 않습니다. LangGraph의 ainvoke가 전체를 비동기로 실행합니다.

# 블로킹 함수를 스레드풀에서 실행
await asyncio.to_thread(_transcribe_sync, audio_path)
await asyncio.to_thread(_select_sync, transcript, duration)
await asyncio.to_thread(_run, ffmpeg_cmd)
